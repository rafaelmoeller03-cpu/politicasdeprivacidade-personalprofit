PERSONAL PROFIT • Política de Privacidade

Política de Privacidade

Esta política explica como o aplicativo PERSONAL PROFIT trata informações dos usuários, profissionais e alunos, garantindo transparência, segurança e respeito à privacidade.

Em caso de dúvidas, entre em contato: mpropersonal1@gmail.com

1. Quais dados são coletados?

  • Dados de cadastro: Nome, e-mail, telefone e informações fornecidas pelo usuário para cadastro de alunos, treinos, avaliações, agenda, financeiro e configurações.
  • Dados de uso: Informações sobre uso dos módulos, datas de agendamento, histórico de treinos, avaliações, exportações e preferências.
  • Dados de pagamento: Apenas para assinaturas premium, o app utiliza o Google Play Billing. Não armazenamos dados bancários ou de cartão.

2. Como os dados são armazenados?

  • Todos os dados são salvos localmente no dispositivo do usuário, utilizando a tecnologia Hive (banco de dados local).
  • O app não envia dados pessoais para servidores externos.
  • O backup e a exportação de dados são realizados apenas por ação do usuário.

3. O que cada módulo faz?

  • Alunos: Cadastro, consulta e edição de informações dos alunos.
  • Treinos: Criação, edição e histórico de treinos personalizados para cada aluno.
  • Avaliações: Registro de avaliações físicas, medidas e evolução dos alunos.
  • Agenda: Gerenciamento de compromissos, sessões, avaliações e eventos, com notificações locais e visualização por dia/mês.
  • Financeiro: Controle de pagamentos, mensalidades, receitas e despesas.
  • Exportação: Permite exportar dados em PDF ou planilhas, apenas por ação do usuário.
  • Compartilhamento: O usuário pode compartilhar informações via WhatsApp, e-mail ou outros apps, sempre de forma manual.
  • Premium: Acesso a recursos exclusivos mediante assinatura, sem coleta adicional de dados.

4. Compartilhamento de dados

  • O app não compartilha dados automaticamente com terceiros.
  • O compartilhamento só ocorre quando o próprio usuário utiliza a função de exportar ou compartilhar.

5. Segurança

  • Os dados ficam protegidos no dispositivo do usuário.
  • Recomenda-se utilizar senha no aparelho para maior segurança.

6. Direitos do usuário

  • O usuário pode editar ou excluir qualquer informação cadastrada a qualquer momento.
  • Para dúvidas ou solicitações, entre em contato: mpropersonal1@gmail.com

7. Crianças e adolescentes

  • O app não é destinado ao uso por menores de 13 anos sem supervisão de um responsável.

8. Alterações nesta política

  • Esta política pode ser atualizada. A versão mais recente estará sempre disponível no app.
